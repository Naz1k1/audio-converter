# 视频转音频转换器 (Audio Converter)

这是一个基于 Spring Boot 和 Vue 的视频转音频的转换的无聊工具，支持多种音频格式的转换。

## 项目架构

项目采用前后端分离的架构：

- 后端：Spring Boot 应用程序
- 前端：Vue + Vite 应用程序

### 后端技术栈

- Spring Boot 2.7.18
- JavaCV 1.5.8（用于音频处理）
- Lombok
- Java Bean Validation

### 前端技术栈

- Vue
- Vite
- Node.js
